# Chef's Menu App

A mobile application designed for a private chef to manage a customizable menu. This app enables the chef to add, view, and remove menu items with ease, providing a seamless experience for guests to view and filter menu options by course. Built using Kotlin on Android Studio, this app enhances the chef's workflow by simplifying menu management and allowing for a dynamic display of average prices per course.

## Features

- **Home Screen with Average Prices**: Displays all menu items categorized by course (Appetizer, Main Course, Dessert) and shows the average price for each course.
- **Add/Remove Menu Items**: A separate screen where the chef can add new menu items or delete existing ones from the menu list.
- **Filter by Course**: Allows guests to filter menu items by course, making it easy to view only specific categories such as starters or main courses.
- **Dynamic Data Storage**: Menu items are stored in an array, with real-time updates to the displayed list.
